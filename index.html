# #欢迎到GitHub页面。

这个自动页面发生器是最简单的方法为你所有的项目创建漂亮的页面。 这里作者页面内容(使用GitHub味减价)(https://guides.github.com/features/mastering-markdown/),选择一个模板由设计师精心设计和发布。 生成页面后,可以查看本地新“gh-pages”分支。 如果你使用GitHub的Mac或Windows GitHub,简单的同步存储库,您将看到新的分支。

# #设计模板

我们精心设计一些漂亮的模板供您使用。 继续点击“继续布局”来浏览它们。 你可以很容易地回到编辑页面之前出版。 发布您的页面后,可以重新打开页面生成器和切换到另一个主题。 你的页面内容将被保留下来。

# #手动创建页面

如果你不喜欢使用自动生成器,推动一个分支叫“gh-pages”存储库创建一个手动页面。 除了支持常规的HTML内容,GitHub页面支持变身怪医,一个简单的博客知道静态网站生成器。 哲基尔很容易创建站点范围的页眉和页脚无需复制他们在每一个页面。 它还提供了智能博客支持和其他高级模板功能。

# #作者和贡献者

你可以@mention GitHub的用户名来生成一个链接到他们的个人资料。 由此产生的“< >”元素将链接到贡献者的GitHub概要文件。 例如:2007年,克里斯Wanstrath(@defunkt)、PJ Hyett(@pjhyett),和汤姆Preston-Werner @mojombo GitHub成立。

# #支持或接触
